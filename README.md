# Harness Engineering — Website Modernization Experiment

A research project (Enkira / NovaServe internship) testing **harness engineering**: does giving a coding agent a harness — tests, verification loops, quality gates — produce better results than running it without one?

The practice task is "modernize this website" on real restaurant sites. Each site is built two ways — with a harness and without — and the outputs are compared, alongside a bake-off of different agent-skill methods.

## What's here

- **`EXPERIMENT.md`** — the full protocol: research questions, controlled design, scoring rubric, and the live results log. **Start here.**
- `arm-a-no-harness/` — builds produced without a harness (the control).
- `arm-b-harness/` — builds produced with a harness.
- `references/` — reference outputs used for grading (not tracked in git).
